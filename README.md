To download RuStore SDK, go to the following url and download Tarball packages (.tgz) for Unity. This extension is only working with .tgz package, and does not support .unitypackage!

Download RuStore SDKs: https://gitflic.ru/company/rustore

RuStore SDK documentation is available here:

https://www.rustore.ru/help

What to do (simply):

1. Download latest RuStore Pay and RuStore Review tarball packages for Unity
2. Add them in Unity Package Manager as tarball packages
3. Add this package to Unity Package Manager by adding the following .git URL:

```url
https://github.com/MirraSDK/SDK5-RuStore-API.git
```

4. Make sure you selected Android platform in Build Settings
5. Go to Toolkit and select RuStore as build configuration
